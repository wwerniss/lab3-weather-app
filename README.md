# Лабораторна робота №3 з ООП  
## Weather App (React + OpenWeatherMap API)

Цей проєкт є реалізацією лабораторної роботи №3 з дисципліни **«Об’єктно-орієнтоване програмування»**.  
Мета — продемонструвати вміння використовувати **зовнішні бібліотеки та фреймворки** у JavaScript на прикладі простого веб-застосунку для перегляду погоди.

Застосунок дозволяє отримувати **поточну погоду за назвою міста** з сервісу [OpenWeatherMap](https://openweathermap.org/).

---

##  Використані технології та бібліотеки

- **React** – бібліотека для побудови інтерфейсу користувача (зовнішній фреймворк).
- **Vite** – інструмент для швидкого старту та збірки React-застосунку.
- **Axios** – HTTP-клієнт для виконання запитів до API.
- **OpenWeatherMap API** – зовнішній сервіс для отримання даних про погоду.
- **JavaScript (ES6+)**, **HTML5**, **CSS3**.

---

##  Функціональність застосунку

- Введення назви міста у текстове поле.
- Надсилання HTTP-запиту до OpenWeatherMap API.
- Отримання та відображення:
  - назви міста;
  - поточної температури (°C);
  - короткого текстового опису погоди;
  - іконки погодних умов.
- Обробка помилок:
  - повідомлення у випадку, якщо місто не знайдено;
  - повідомлення у випадку проблем з мережею або API.
